CreateMainInfoFromDataRow(mainInfoRow) method

Summary

This method create a MainInfo object from a datarow. It is a method created to be used for all methods that return MainInfo object or MainInfo object array so we can recycle code. This polymorphic method is designed for the call in the creation of an array of MainInfo objects, avoiding hundreds of queries by means of the dataset passed as a parameter since we pre-load the relational tables.
